# Workout Tracker

App for tracking workouts. Built for CSC 437.

## How to run it
1. Clone the repo and run `npm install`
2. Create `packages/server/.env` with the MongoDB credentials and a TOKEN_SECRET
3. `npm run build -w packages/app`
4. `npm run start:app -w packages/server`

Register a new account on the register page, and you'll be redirected to the main page.
